## Setup

```sh
$ git clone https://github.com/Tur-4000/php-project-lvl2.git

$ make install
```

## Run tests

```sh
$ make test
```

## Usage

```sh
gendiff [--format <fmt>] FIRST_FILE SECOND_FILE
```
